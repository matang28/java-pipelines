## How to install:

Add jitpack.io maven repository to your build.gradle file:
```
repositories {
    mavenCentral()
    maven { url 'https://jitpack.io' }
}
```

then add the dependency to your build.grade file:
```
dependencies {
    ...
    compile 'com.github.matang28:java-pipelines:1.0'

}
```
